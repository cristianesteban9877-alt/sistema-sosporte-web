Introducción al Sistema

¿Cómo nació la idea del proyecto?

El Sistema de Soporte Sagrado Corazón Al. Cal (SsSC) trata sobre una página web y aplicación móvil diseñadas para administrar el soporte técnico de la escuela. Ofrecemos una visualización de dispositivos ordenada por aulas y un sistema de reportes que otorga una actualización para la visión de estado de las computadoras.
Con este sistema se pueden registrar, editar, eliminar y controlar todos los dispositivos de todas las aulas con permisos distribuidos entre cinco tipos de usuarios: Administrador, Soporte técnico, Profesor, Profesor de Técnica y Alumno.
Luego de haber formado el grupo de proyecto teníamos que pensar un proyecto que se pueda aplicar a una página web y aplicación móvil además de que debía ser útil, este proyecto lo llevaríamos a cabo durante lo que restara del ciclo lectivo 2025. Tras debatirlo con el grupo decidimos crear el sistema de soporte con el objetivo de crear algo que pueda utilizarse en años posteriores. Una vez la idea fue aprobada por los profesores, el grupo se comprometió para llevar el desarrollo a cabo.

¿Cómo funciona el sistema?

Este sistema permite visualizar el estado de los dispositivos en tiempo real, generar reportes detallados sobre problemas específicos, asignar o desasignar gabinetes o monitores, así como registrar, eliminar o editar la información de usuarios, gabinetes, monitores y aulas.
El sistema está compuesto por dos plataformas principales: una página web y una aplicación móvil. Ambas versiones ofrecerán las mismas funcionalidades, permitiendo a los usuarios realizar las mismas acciones desde cualquier dispositivo. La plataforma estará diseñada bajo una arquitectura centralizada, lo que garantiza que cualquier modificación o actualización realizada desde uno de los entornos (web o móvil) se refleje automáticamente en el otro, gracias a la conexión a una base de datos común. Esta sincronización en tiempo real asegura la coherencia de la información y una experiencia de usuario fluida y continua.
